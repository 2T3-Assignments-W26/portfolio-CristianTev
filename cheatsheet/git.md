Title: Cheat Sheet

&nbsp; To initialize a new Git repository, navigate to your project directory and run the following command: git init

if it is in a repo use this for iniciate t in a specific directory git init <directory>

After initializing the repository, you need to add your project files to the repository and make the first commit. Here are the steps:

Add all files to the staging area:

git add

Commit the files with a message:

git commit -m "Initial commit"

get the status of a repo : git status

how to clone

Go to GitHub on the browser:

Login to your account.

Click on your profile icon (right top corner) >> Repositories

Click on the repo that you want to clone

Click on the Code button.

Choose Clone with HTTPS option.

Copy the URL.

Go back to the terminal.

Make sure you are still in the same location of the empty folder you have created earlier.

&nbsp;

To clone use the command:

git clone \[HTTPS URL copied from GitHub]

You may get a prompt to get authenticated: choose Sign in with your browser.

&nbsp;

Remember you clone, the remote repo is already added.
